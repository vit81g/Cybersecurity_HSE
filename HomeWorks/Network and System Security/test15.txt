8. Какие утверждения верны относительно доступа к маршрутизаторам Cisco?

Маршрутизаторы Cisco поддерживают сохранение паролей учетных записей пользователей в виде хэшей. Рекомендуемый метод хеширования – secret 8

Маршрутизаторы Cisco поддерживают несколько уровней привилегий пользователей. Уровень 1 обеспечивают ограниченный ввод команд

9. Какие утверждения верны об аутентификации в протоколе SSH?

При использовании аутентификации по ключам необходимо обеспечить сохранность закрытого ключа клиента SSH

Для аутентификации по ключам, необходимо выполнить загрузку открытого ключа клиента на управляемое устройство

SSH поддерживает аутентификацию по сертификатам. Для подключения , на устройстве должен быть установлен сертификат доверенного центра сертификации
