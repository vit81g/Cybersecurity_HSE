Тест 12. DNS

Какие утверждения верны в отношении для одноранговых и клиент-серверных приложений (возможно несколько вариантов):

для клиент-серверных приложений, используется централизованный подход к безопасности

одноранговые приложения, являются распределенными и устойчивы к сбоям

клиент-серверная модель поддерживается большинством приложений

2. Специалист технической поддержки решает проблему пользователя с недоступностью некоторых сетевых ресурсов. Пользователь сообщает, что в окне веб-браузера открываются некоторые корпоративные ресурсы, но большинство не открывается. Специалист проанализировал ярлыки доступных приложений и выяснил, то ссылки представлены в виде IP адресов. Ресурсы, представленные в виде FQDN, не открываются. С чем связана проблема?
 (возможно несколько вариантов)

DNS-сервер заблокирован корпоративным межсетевым экраном

адрес DNS сервера не корректен

3. Корпоративный сервер DNS сконфигурирован с использованием передачи запросов на корневые ссылки (root hints). Какие утверждения верны для описанной конфигурации? 
 (возможно несколько вариантов)

корневые ссылки нельзя использовать клиенту для разрешения имен, только прочие DNS -серверы могут формировать запросы на корневые ссылки

любое корректное и существующее доменное имя Интернет может быть разрешено с помощью корневых ссылок

в случае отсутствия запрашиваемой зоны DNS на локальном сервере, запрос будет перенаправлен на корневые ссылки

4. Выберите проблемы безопасности характерные для DNS 

DNS-туннели позволяют злоумышленникам организовать канал утечки данных

подмена DNS ответов позволяет выполнить отравление DNS-кэша клиентов и перенаправить клиента на мошеннический интернет-ресурс

DNS-серверы уязвимы к атаке типа «UDP amplification», способной направить большой поток UDP ответов на атакуемую системы


5. Сервер DNS сконфигурирован в качестве кеширующего сервера. Что из нижеперечисленного неверно для указанной конфигурации? 

каждый ответ кеширующего DNS сервера является авторитативным

6. Выберите верное утверждение о системе DNS:

система DNS поддерживает иерархическую систему имен, каждый хост в системе DNS содержит FQDN

7. рабочим станциям назначается IP адрес в сети 169.254.0.0/16

8. Какой порядок сообщений протокола DHCP является корректным?

Discover, Offer, Request, Ack - DORA

9. Системный инженер выполняет проектирование сетевой инфраструктуры. Планируется установка централизованного сервера DHCP, обслуживающего несколько филиалов организации. В каждом филиале планируется несколько подсетей и наличие серверов не планируется. Выберите оптимальное техническое решение?

на сетевом оборудовании филиала организовать DHCP Relay для каждой из сетей. DHCP Relay настроить на использование IP адреса сервера DHCP

10. Какие утверждения верны для протоколов HTTP и HTTPS?

использование аутентификации клиента HTTPS является опциональным

набор криптографических протоколов защиты информации определяется на этапе установки соединения HTTPS

HTTPS поддерживает аутентификацию как с помощью предварительных ключей PSK, так и с помощью сертификатов
