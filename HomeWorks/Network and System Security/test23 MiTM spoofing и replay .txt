Тема 23. MiTM, spoofing и replay

1. К какой категории относится атака через посредника (MITM)?

доступ

2. Выберите типы сетевых атак уровня L2 (возможно несколько вариантов ответа)

атака с переходом VLAN
атаки ARP
атаки подмены

3. Какая процедура рекомендуется для снижения вероятности спуфинга ARP?

включение отслеживания DHCP для выбранных VLAN

4. Какой тип атаки возникает, когда злоумышленнику удаётся подслушать разговор между двумя IPS-телефонами?

человек посередине

5. Выберите признаки спуфинга в электронной почте (возможно несколько вариантов ответа)

тайпсквоттинг

ошибки в написании адреса электронной почты

6. Какие рекомендации верны для борьбы с атакой «человек посередине»? (возможно несколько вариантов ответа)

использование антивирусных программ

использование файрволла

использование мультифакторной авторизации


7. Злоумышленник объявляет поддельный MAC-адрес сетевым хостам, пытаясь подделать шлюз по умолчанию. Какую команду необходимо использовать на сетевых узлах, чтобы узнать MAC-адрес для доступа к шлюзу по умолчанию?

ARP -a


8. Как можно нейтрализовать атаку DHCP-спуфинга?

путём реализации DHCP snooping на доверенных портах


9. Какая основная цель атаки MITM?

доступ к конфиденциальным данным

10. Какой метод используется для борьбы с атакой повторного воспроизведения?

использовать IPSec
