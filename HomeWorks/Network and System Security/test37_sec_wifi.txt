 Тема 37. Защита беспроводных сетей

1. Злоумышленник, анализируя конфигурации сети Wi-Fi, выполнил проксирования сессии администратора, завладел паролем контроллера AC. Далее выполнил смену настроек сети, из-за чего пользователи не смогли подключиться. В какой плоскости сети была обнаружена уязвимость?

Плоскость управления

2. Злоумышленник, анализируя конфигурации сети Wi-Fi, обнаружил уязвимый протокол защиты WEP. После анализа некоторого количества шифрованных пакетов с помощью беспроводного сниффера был вычислен ключ WEP,  благодаря которому злоумышленник смог подключиться к сети. В какой плоскости была обнаружена уязвимость?

Плоскость контроля

3. Злоумышленник выполнил подключение к открытой беспроводной сети и использует сканер безопасности для выявления открытых уязвимых сервисов у пользователей сети с целью последующей эксплуатации. В какой плоскости беспроводной сети была использована уязвимость?

Плоскость данных/передачи

4. При выполнении тестирования на проникновение беспроводной сети специалист по информационной безопасности создал беспроводную сеть, SSID которой копирует SSID корпоративной сети. Несколько корпоративных пользователей подключились к этой сети,
введя PSK, которым завладел специалист. Какие меры помогут в дальнейшем избежать таких атак?
(Возможно несколько вариантов)

Использовать WIDS/WIPS

Скрыть SSID корпоративной сети

Использовать изоляцию пользователей в беспроводной сети

Усовершенствовать способ аутентификации пользователей, использовать EAP

Повысить уровень осведомлённости пользователей беспроводной сети о вопросах возможных атак


5. Системный инженер производит проектирование нового развёртывания сети Wi-Fi. В качестве метода контроля доступа предлагается использование WPA3. Какие преимущества обеспечивает этот метод?
(Возможно несколько вариантов)

Отсутствие подтверждённых уязвимостей

Усовершенствованные механизмы шифрования

Наличие криптографической защиты в сетях с открытой аутентификацией

Автоматическая защита плоскости управления

6. В беспроводной сети используется метод контроля доступа WPA2-PSK. Выберите верные утверждения об этой сети. (Возможно несколько вариантов)

При использовании WPA2-PSK в крупных корпоративных сетях возникает проблема сохранности конфиденциальности ключа PSK

Этот метод защиты подходит для домашних сетей

Обладая ключом PSK, злоумышленник имеет возможность перехвата трафика пользователей и дешифровки

7. Какие технологии защиты плоскости управления являются наиболее эффективными? (Возможно несколько вариантов)

Использование архитектуры сети управления, предусматривающей отделение от сети передачи
данных

Использование защищённых протоколов управления SSH и HTTPS


8. Выберите преимущества изоляции беспроводных клиентов друг от друга. (Возможно несколько вариантов)

Предотвращается атака по сканированию портов и уязвимостей злоумышленником, подключённым
к беспроводной сети

Предотвращается горизонтальное распространение вредоносного программного обеспечения

Предотвращается нарушение политик безопасности в части передачи файлов между клиентами в одноранговой сети

9. Выберите верные утверждения об уязвимостях и защитных мерах, связанных с CAPWAP. (Возможно несколько вариантов)

Злоумышленник может вызвать отказ в обслуживании AC и всей беспроводной сети, отправив специально подготовленный пакет CAPWAP

IPSec VPN является эффективным средством защиты CAPWAP в случае развёртывания точек доступа в филиалах

DTLS является эффективной мерой защиты от модификации данных, передаваемых через
CAPWAP

10. При проектировании крупной беспроводной сети организации с филиалами используется WPA2-EAP. Что необходимо учитывать при дизайне этой сети? (Возможно несколько вариантов)

Исключить единую точку отказа в части AAA

Использовать режим EAP и поддерживать соответствующую инфраструктуру ключей и
носителей

Применять сетевые средства защиты информации: NGFW и NGIPS
