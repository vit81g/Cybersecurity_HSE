Тест по Теме 13  AD

1. Какой из перечисленных протоколов файлового доступа является основным для сетей Windows?

CIFS/SMB

2. Пользователь в ОС Windows изменил настройки приложения, использующегося для открытия файлов PDF. В какой из веток реестра были сохранены настройки?

HKEY_CLASSES_ROOT

3.  Администратор выполнил смену пароля локального пользователя на одном из компьютеров в сети Windows. Где были сохранены изменения?

в файле SAM на локальном компьютере

4. Архитектор ИТ выполняет оценку облачных сервисов для своей компании. Выявлена потребность в использовании офисного пакета онлайн и объектного хранилища для совместной работы пользователей. Какую из услуг следует рекомендовать?

SAAS

5. Какой сервис Active Directory отвечает за безопасную и надежную аутентификацию в домене?

Kerberos

6. Сетевой администратор выполняет установку сервера Exchange в корпоративной сети Windows. Exchange использует ресурсы каталога Active Directory и требует внести новые атрибуты в домене – новые поля в БД для хранения информации. Какой раздел Active Directory изменяется в рамках установки Exchange?

Schema

7. На каждом из контроллеров домена, доступна функция создания учетных записей пользователей. Какая из FSMO ролей обеспечивает уникальность идентификаторов создаваемых пользователей?

RID

8. Что определяет функциональный уровень домена?

минимальную версию операционной системы последующих контроллеров домена в домене

9. В физической структуре службы каталогов Active Directory представлены сайты и контроллеры домена. Выберите верные утверждения? (возможно несколько вариантов)

принадлежность к сайту позволяет определить правила репликации между контроллерами домена

признак принадлежности контроллера домена к сайту позволяет клиентам найти ближайший контроллер домена

10. Какие утверждения верны для контроллера RODC? (возможно несколько вариантов)

не позволяет вносить изменения в каталог

обычно используется на удаленных площадках, где физическая безопасность серверов не может быть обеспечена
