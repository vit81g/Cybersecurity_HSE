3. Выберите ошибочные утверждения о конфигурации STP коммутируемой сети (возможно несколько вариантов):

Настройки по умолчанию работоспособны не во всех ситуациях. В частности, в случае совпадения Bridge Priority, коммутаторы не смогут выбрать Root Bridge и порты не перейдут в состояние передачи

При выборе Root Bridge, необходимо изменять системные MAC адреса коммутаторов так, чтобы у кандидата был наибольший по значению MAC

6. Какие утверждения верны в отношении стекирования коммутаторов (возможно несколько вариантов):

Стекирование коммутаторов требует либо выделенных специализированных интерфейсов стекирования для объединения коммутаторов между собой, либо портов SFP+/QSFP/QSFP+

При подключении к IP адресу управления стека коммутаторов, обработку команд управления производит мастер-коммутатор

При использования стекирования, есть возможность подключения агрегированного LACP канала, оканчивающегося на разных коммутаторах стека.
