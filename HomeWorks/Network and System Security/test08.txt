Тест по Теме 8. Динамическая маршрутизация. Протоколы внутреннего шлюза

1. Выберите верные утверждения о статической маршрутизации (возможно несколько вариантов):

статические маршруты задаются администратором и не передаются между устройствами в сети

в общем случае, статические маршруты не обеспечивают резервирование каналов связи

статические маршруты являются более безопасными. Сторонние маршрутизаторы не могут манипулировать протокольными сообщениями.

2. Выберите верные утверждения о протоколе RIP  (возможно несколько вариантов):

использует дистанционно-векторный алгоритм в основе механизмов расчетов маршрутов

метрика маршрутов, на устройствах Cisco по умолчанию, равна 120

существуют две версии протокола RIP для IPv4

3. Маршрутизатор поддерживает несколько протоколов маршрутизации. При одновременном запуске нескольких протоколов маршрутизации, используется порядок их появления в таблице маршрутизатора. Выберите корректный порядок появления в таблице (от большего приоритета к меньшему):

Static – eBGP-OSPF-RIP-iBGP

4. Выберите верное утверждение:

протокол BGP, как правило может использоваться для маршрутизации в сети Интернет

5. Выберите верное утверждение о протоколе RIPv2:

протокол RIPv2 поддерживает автоматическую суммаризацию по классовой границе и ручную суммаризацию

6. Выберите верные утверждения о протоколах RIP и OSPF (возможно несколько вариантов):

протокол RIP использует алгоритм Дейкстры для определения оптимального маршрута

метрикой протокола OSPF является стоимость маршрута – величина, обратно пропорциональная пропускной способности интерфейсов. Метрика RIP – количество транзитных переходов

OSPF – протокол, подходящий для крупных распределенных сетей и поддерживает области. Протокол RIP имеет ограничения для использования в современных сетях

7. Выберите верные утверждения о маршрутизаторах ABR и ASBR (возможно несколько вариантов):

внешние маршруты, поступающие с ASBR, маркируются в таблицах OSPF как O*E1 и O*E2, в зависимости от типа маршрутов

маршрут по умолчанию может быть передан от ASBR к другим маршрутизаторам OSPF

8. Какое утверждение в отношении областей OSPF является верным:

использование областей позволяет сегментировать базу данных каналов связи (LSDB). Только маршрутизаторы ABR содержат в себе LSDB нескольких областей

9. Выберите ошибочные утверждения о метрике протокола OSPF (возможно несколько вариантов):

по умолчанию, интерфейсы с пропускной способностью 100 Мбит/с и выше, имеют одинаковую пропускную способность.

команда auto cost reference-bandwidth используется для переоценки стоимостей интерфейсов OSPF
