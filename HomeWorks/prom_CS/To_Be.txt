АСУ ТП (разработка ПО на аутсорсе)):
опрос всех микроконтроллеров и технологического оборудования АСУ ТП -> 
отправка данных опроса на log-коллектор -> 
нормализация данных для SOC ->
отправка данных по зашифрованному каналу в SOC

Объект КИИ 2 категория:
1. архитектура систмы защиты с привлечением интегратора
1.1 DMZ зона (NGFW, WAF)
1.2 импортозамещение сетевого оборудования
1.3 внедрение СЗИ на конечных точках (АВЗ, НСД, EDR/XDR)
1.4 внедрение сетевых СЗИ (NGFW, WAF, сегментация сетей, NAD)
1.5 внедрение PAM (управление доступом)
1.6 внедрение резерновного копирования
1.7 внедрение системы контроля изменений 
1.8 внедрение СКУД, 2FA (НСД)

2. настройка систем защиты

3. орагнизация управления СЗИ на объекте

4. разработка документации
4.1 управление доступом
4.2 управление изменениями
4.3 управление СКУД
4.4 реагирование на инциденты

5. настройка сбора логов с СЗИ -> 
отправка логов СЗИ на log-коллектор -> 
нормализация данных для SOC ->
отправка данных по зашифрованному каналу в SOC

6. обучение персонала на местах, обучение администратора ИБ
