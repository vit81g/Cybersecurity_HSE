Тест по теме 19
Тема 19. Атаки на локальную сеть: атаки канального уровня

1. Какой тип атаки с переключением VLAN можно предотвратить, назначив неиспользуемую VLAN собственной VLAN?
Двойная маркировка VLAN

2. Какой протокол следует отключить, чтобы нейтрализовать атаку с переключением VLAN?

DTP

3. Какая контрмера безопасности эффективна для предотвращения атак переполнения таблицы CAM?

Безопасность порта

4. Какую функцию безопасности следует включить, чтобы злоумышленник не переполнил таблицу MAC-адресов коммутатора?

Port Security

5. Выберите функцию, предотвращающую изменение топологии связующего дерева путём блокировки порта, который получает вышестоящий BPDU?

Root Guard

6. Как будет себя вести коммутатор в результате успешной атаки на CAM-таблицу?

Коммутатор будет пересылать все полученные кадры на все остальные порты

7. Какие дополнительные меры безопасности необходимо включить вместе с IP Source Guard для защиты от подмены адресов?

Отслеживание DHCP

8. Выберите методы подавления атак с переключением VLAN

Включение транкинга вручную
Установка для собственной VLAN неиспользуемой VLAN
Отключение DTP

9. Какой механизм стабильности STP используется для предотвращения преобразования мошеннического коммутатора в корневой коммутатор?

Root Guard

10. Какие преимущества в безопасности даёт включение защиты BPDU на интерфейсах с поддержкой PortFast?

Предотвращение добавления в сеть мошеннических коммутаторов
